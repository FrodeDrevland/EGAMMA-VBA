EGAMMA_DIST(x, alpha, beta, delta, cumulative):

Calculate the expanded gamma distribution.

Parameters:

  • x – Required. The value at which the function is evaluated.

  • alpha – Required. The shape parameter of the distribution.

  • beta – Required. The scale parameter of the distribution. Negative beta entails a left skewed distribution.

  • delta – Required. The location parameter of the distribution.

  • cumulative – Required. A logical value that determines the form of the function. If cumulative is TRUE, GAMMA.DIST returns the cumulative distribution function; if FALSE, it returns the probability density function.

Returns:

The value of the (cumulative) gamma distribution.

Return type:

float

# The functions EGAMMA_TPE_TO_PARAMS, FindAlphaAtModeEqualsProbability, FindAlpha, and EGAMMA_FIT_TO_PARAMS are complex and would require similarly detailed docstrings. Each function should have a description of its purpose, parameters, return value, and any exceptions.

EGAMMA_FIT_TO_PARAMS(*args):

Fit the parameters of the expanded gamma distribution to a given set of data points.

This function estimates the alpha, beta, and delta parameters of the expanded gamma distribution, based on the provided data using the method of moments. It uses the skewness of the data to estimate alpha, then calculates beta using the standard deviation, and finally computes delta from the mean, alpha, and beta.

Parameters:

args – A variable number of arrays or lists containing numerical data points. These are combined into a single dataset for the analysis.

Returns:

A list containing the estimated parameters [alpha, beta, delta] of the extended gamma distribution.

Raises:

ValueError – If the input data does not allow for a proper fit, e.g., if it leads to divide-by-zero errors.

EGAMMA_TPE_TO_PARAMS(low, likely, high, low_probability=0.1):

Estimate the parameters of the expanded gamma distribution based on a three-point estimate.

This function uses a low, likely, and high estimate to determine the parameters of the extended gamma distribution.

Parameters:

  • low – The low estimate of the distribution.

  • likely – The most likely estimate of the distribution.

  • high – The high estimate of the distribution.

  • low_probability – The probability of the low estimates (default is 0.1).

Returns:

A list containing the estimated parameters [alpha, beta, delta] of the extended gamma distribution.

Raises:

ValueError – If the input values are inconsistent (e.g., low > likely, high < likely, or all values are equal).
